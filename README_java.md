# Java Number Guessing Game

This is a Java implementation of the number guessing game.

Files

Compile and run

```bash
javac "task 1/GuessingGame.java"
java -cp "task 1" GuessingGame
```

Notes

- Enter blank input at prompts to accept defaults (minimum=1, maximum=100, attempts=0 for unlimited).
- Enter `0` for max attempts to allow unlimited guesses.
- After each round you'll be asked whether to play again.
- A summary is printed when you exit the game.
